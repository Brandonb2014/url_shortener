Welcome to my URL shortener!

Use a program such as Postman to send a POST request to:

http://localhost:3000/shorten

with a JSON body like so:

{ "url": "http://www.thisisareallylongurlthatneedsashortcode.com" }

it will then give you a shortened URL you can visit to be redirected to your long URL.

NOTE: You can also visit:

http://localhost:3000/api/shortUrls

to get a list of currently shortened URLs.
